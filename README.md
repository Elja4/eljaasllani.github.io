## Projects

### Castle Adventure Game

**Technologies:** C#, Unity

A 3D adventure game where the player controls a girl navigating through a castle filled with obstacles and challenges. The game focuses on movement, obstacle avoidance, and interactive gameplay.

**Features**

* Character movement and controls
* Obstacle-based gameplay
* Multiple castle environments
* Interactive 3D experience
